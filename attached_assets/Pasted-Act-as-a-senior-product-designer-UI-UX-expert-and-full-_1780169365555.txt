Act as a senior product designer, UI/UX expert, and full-stack ecommerce developer.

Create a modern, clean, minimalist ecommerce website.

The goal is not to create an experimental or artistic website.

The goal is to create a beautiful ecommerce website that feels trustworthy, premium, and easy to use.

Design Principles:

- Minimalist
- Modern
- Clean
- Professional
- Premium
- Conversion-focused
- Fast and responsive

Avoid:

- Overly complex layouts
- Excessive animations
- Glassmorphism everywhere
- Futuristic aesthetics
- Neon colors
- Gaming-inspired UI
- Visual clutter

Theme System:

Create both Light Mode and Dark Mode.

Light Mode:
- Soft white backgrounds
- Neutral grays
- High readability
- Clean shadows

Dark Mode:
- Rich charcoal backgrounds
- Soft contrast
- Elegant appearance
- No pure black backgrounds

Users can switch themes manually.

Remember theme preference using local storage.

Before generating code:

1. Create complete design system.
2. Define color tokens for both themes.
3. Define spacing system.
4. Define component architecture.
